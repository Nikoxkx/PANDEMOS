> *"In the age of misinformation, the most radical act is clarity."*

Pandemics don't wait. Neither should awareness. **PANDEMOS** is an open-source disease surveillance and public health education platform built for researchers, students, educators, and informed citizens who refuse to be caught off guard. It aggregates data from the world's most trusted health institutions and presents it with precision, transparency, and zero noise.

---

## ⚠️ Disclaimer

> **PANDEMOS is strictly for educational and informational purposes.**
> It is **not** a substitute for professional medical advice, diagnosis, or treatment. In a health emergency, always defer to qualified healthcare providers and official public health authorities.

---

## What PANDEMOS Does

PANDEMOS sits at the intersection of **surveillance intelligence**, **public health literacy**, and **open data**. It is not a news aggregator. It is not a panic machine. It is a structured, sourced, and sober lens on the state of infectious disease — globally, historically, and right now.

---

## Core Modules

### 🌍 War Room — Global Outbreak Intelligence

> *The operational heart of PANDEMOS.*

A live-updating, interactive global surveillance dashboard built for situational awareness.

- **Interactive world map** overlaid with real-time outbreak data from official sources
- **Filter system** by disease type, transmission route, and severity classification
- **Source-attributed outbreak cards** — every data point traces back to its origin
- **Historical timeline scrubber** — visualize how outbreaks evolve and spread over time
- Built on a custom SVG map with NASA satellite imagery as the base layer

---

### 🧬 Disease Catalog — The Pathogen Index

> *A clinical-grade reference library, built for everyone.*

An exhaustive, rigorously structured database covering the full spectrum of infectious disease threats — from endemic to emergent.

**Diseases covered include (but are not limited to):**

| Category | Diseases |
|---|---|
| Viral Hemorrhagic | Ebola, Marburg, Yellow Fever |
| Respiratory | COVID-19, H5N1 Avian Influenza, Tuberculosis |
| Vector-Borne | Dengue, Malaria, Plague |
| Emerging | Mpox, Nipah, Hantavirus |
| Chronic Infectious | HIV/AIDS, Hepatitis B & C, Rabies |
| Vaccine-Preventable | Measles, Cholera, Meningitis |

**Every disease entry includes:**

- Quick-reference statistics — R₀, case fatality rate, incubation period, transmission route
- Body systems affected with detailed pathophysiology breakdowns
- Symptom progression mapped across clinical phases
- Discovery and outbreak history with dated timelines
- Current treatment protocols and vaccine availability
- Direct citations to peer-reviewed literature and official health body guidance

---

### 📚 Educational Center — Public Health Literacy

> *Because understanding disease is the first line of defense.*

A structured, interactive learning environment covering the foundational concepts of epidemiology and public health — written for clarity, not condescension.

**Learning Modules:**

| Module | Description |
|---|---|
| 🦠 Understanding Pathogens | Viruses, bacteria, fungi, prions, and parasites — what they are and how they operate |
| 💉 How Vaccines Work | mRNA, viral vector, inactivated, and live-attenuated mechanisms explained |
| 📈 Reading Epidemic Curves | R₀ interpretation, curve morphology, lag indicators, and suppression signals |
| 🔍 Source Credibility | A practical framework for evaluating health information in a noisy media environment |
| 🐾 Zoonotic Spillover | Reservoir hosts, interspecies transmission, and why most pandemics start in animals |
| 🧫 Antimicrobial Resistance | The slow-motion crisis threatening the foundation of modern medicine |

**Also includes:**
- An interactive **pandemic history timeline** spanning centuries of outbreak events
- A comprehensive **epidemiology glossary** with plain-language definitions

---

### 🔗 Sources & Credibility — The Transparency Layer

> *Every claim. Every source. Every tier.*

PANDEMOS operates on a **5-tier credibility rating system**, making it explicit where data comes from and how much weight it carries. No anonymous sourcing. No algorithmic aggregation without attribution.

**Tier 1 — Institutional Health Authorities**
WHO · CDC · ECDC · Africa CDC · PAHO

**Tier 2 — Peer-Reviewed Scientific Literature**
The Lancet · NEJM · Nature · Science · JAMA

**Tier 3 — Genomic & Epidemiological Surveillance**
GISAID · Nextstrain · Our World in Data

**Tier 4 — Humanitarian Field Intelligence**
MSF (Médecins Sans Frontières) · UNICEF · GAVI

**Tier 5 — National Health Ministries**
Verified government health authority reports from jurisdictions worldwide

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 19 with TypeScript |
| Build Tool | Vite |
| Styling | Tailwind CSS v4 |
| State Management | Zustand |
| Maps | Custom SVG · NASA satellite imagery |
| Deployment | — |

**Design Philosophy:**
PANDEMOS uses an Apple-inspired **glass morphism** visual language — semi-transparent surfaces, backdrop blur, disciplined typography, and a dark/light mode system — designed to make dense health data readable without sacrificing seriousness or precision. Fully responsive across all screen sizes.

## Getting Started

```bash

git clone https://github.com/your-username/pandemos.git
cd pandemos

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## Data Sources

All data is sourced exclusively from verified, official public health institutions and peer-reviewed literature:

- World Health Organization (WHO)
- U.S. Centers for Disease Control and Prevention (CDC)
- European Centre for Disease Prevention and Control (ECDC)
- Africa Centres for Disease Control and Prevention (Africa CDC)
- Pan American Health Organization (PAHO)
- National health ministries worldwide
- Peer-reviewed journals including The Lancet, NEJM, Nature, and Science

PANDEMOS does not manufacture, estimate, or editorialize health data. If a source says it, we cite it. If we can't cite it, it doesn't appear.

Contributions are welcome — and genuinely needed. Public health is a collective responsibility, and so is this platform.

1. Read the [Contributing Guidelines](./CONTRIBUTING.md)
